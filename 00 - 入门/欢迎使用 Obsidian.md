# 欢迎使用 Obsidian

这是 **IFRS + US GAAP 双准则知识库**，主要在 **Cursor** 中用于准则检索与会计问题分析。

## 目录结构

| 目录 | 说明 |
|------|------|
| **00 - 入门** | 本指南 |
| **02 - 项目** | 实务 Q&A 项目笔记 |
| **03 - 知识库** | IFRS / US GAAP 准则全文（核心） |
| **04 - Cursor协作** | Agent 任务交接（可选） |

> 完整说明见根目录 [[README]]。

## 在 Cursor 中提问

直接在 Cursor 聊天中提出 IFRS 或 US GAAP 问题，例如：

> 「IFRS 16 下短期租赁豁免如何判断？按项目编写说明写。」

Agent 会在 `03 - 知识库/` 中检索准则，并在 `02 - 项目/` 生成结构化笔记。

## Obsidian 快捷操作

| 操作 | 快捷键 |
|------|--------|
| 快速切换笔记 | `Ctrl/Cmd + O` |
| 命令面板 | `Ctrl/Cmd + P` |
| 打开图谱 | `Ctrl/Cmd + G` |
| 搜索所有笔记 | `Ctrl/Cmd + Shift + F` |
