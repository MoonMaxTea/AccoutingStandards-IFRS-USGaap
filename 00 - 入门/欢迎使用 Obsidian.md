# 👋 欢迎使用 Obsidian

这是你的个人知识库，由 **Codex** 帮你初始化配置。

## 目录结构

- **00 - 入门**：新手指南和配置说明
- **01 - 日记**：每天一条日记，按日期自动归档
- **02 - 项目**：项目相关笔记、计划和跟踪
- **03 - 知识库**：长期积累的知识卡片

## 快捷操作

| 操作 | 快捷键 |
|------|--------|
| 快速切换笔记 | `Ctrl/Cmd + O` |
| 命令面板 | `Ctrl/Cmd + P` |
| 插入模板 | `Ctrl/Cmd + T`（需 Templater 插件） |
| 打开图谱 | `Ctrl/Cmd + G` |
| 搜索所有笔记 | `Ctrl/Cmd + Shift + F` |

## 已配置的插件

安装后请在 `设置 > 第三方插件` 中启用。推荐安装：

- **Dataview** — 用 SQL 式查询展示笔记数据
- **Templater** — 强大的模板引擎
- **Calendar** — 日历视图浏览日记
- **Kanban** — 看板管理任务
- **Excalidraw** — 手绘风格图表
- **Obsidian Git** — Git 备份同步
